# 🛒 Mini E-Commerce Application

A full-stack e-commerce web application built using **React (Frontend)** and **Node.js + Express (Backend)**.  
This project demonstrates real-world application flow including product browsing, category filtering, and cart management.

---

## 🚀 Features

- 📦 Product listing with categories
- 🧭 Sidebar category filtering (Electronics, Beauty, Fashion, etc.)
- 🛒 Add to Cart functionality
- 🔢 Increase / Decrease product quantity
- ❌ Remove items from cart
- 💰 Dynamic total price calculation
- 🖼️ Product images display
- 📂 Clean UI with responsive layout

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6)
- CSS (inline styling)

### Backend
- Node.js
- Express.js
- CORS

How to Run Locally

### 1️⃣ Clone the repository


git clone https://github.com/Shivanichowdary24/Mini-ecommerce-app.git

Run Backend


cd backend
npm install
node server.js


Backend will run on:

http://localhost:5050


---

### 3️⃣ Run Frontend


cd ecommerce-frontend
npm install
npm start

Frontend will run on:

http://localhost:3000
